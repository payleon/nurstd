# Published Tests Directory

Place your NCLEX HTML test files in this directory.

## How it works

1. All HTML files in this directory will automatically appear in the test list
2. File names should follow this format: `test_name.html`
3. The test title will be generated from the file name (spaces will replace underscores)
4. To add a new test, simply place the HTML file in this directory

## Example

For a file named `nclex_testing_1.html`, the platform will display it as "Nclex Testing 1" in the test list.
